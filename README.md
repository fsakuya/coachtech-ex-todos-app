# Todo管理アプリ
Todoを入力／更新／削除することでTodo管理を円滑に実施する
![image](https://user-images.githubusercontent.com/114271510/222890849-c470e7cf-0552-4e28-9976-299eba4f5665.png)

## 作成した目的
Todo管理を行う為

## 機能一覧
- Todo一覧取得
- Todo作成
- Todo更新
- Todo削除

## 使用技術（実行環境）
- Laravel　8.x
